# DQN-агенты для RoboCup 2D Soccer

Цель: обучение футбольных агентов с помощью глубокого обучения с подкреплением в симуляторе RoboCup 2D.

---

## Идея

Обычные агенты в RoboCup играют по заранее написанным правилам. В этом проекте агенты учатся сами — через тысячи матчей против команды helios они постепенно понимают, когда бить по воротам, когда пасовать, а когда просто держать мяч.

Каждый агент управляет одним игроком на поле и обучается независимо от остальных. Когда один агент научился хорошо играть на своей позиции, его знания передаются следующему агенту как стартовая точка — transfer learning.

---


## Архитектура

```
rcssserver (симулятор)
    ↕
helios-base (C++) — логика игры, FSM
    ↕  pybind11 (внутрипроцессный вызов)
Python / PyTorch — нейросеть, обучение
```

Python нейросеть встроена прямо в C++ процесс через pybind11.

## Возможности агента

Агент выбирает одно из 8 **макро-действий**:

| # | Действие | Когда |
|---|----------|-------|
| 1 | Удар по воротам | Есть возможность забить |
| 2 | Пас | Рядом есть свободный партнёр |
| 3 | Дриблинг | Можно обойти соперника |
| 4 | Выбивание | Нужно убрать мяч из опасной зоны |
| 5 | Удержание | Выиграть время |
| 6 | Перехват | Мяч рядом и можно забрать |
| 7 | Блокировка | Помешать сопернику |
| 8 | Позиционирование | Занять тактическую позицию |

## Видимость агента

- Расстояние и угол до мяча
- Расстояние и угол до ворот соперника
- Своя позиция на поле (x, y)
- Расстояния и углы до 5 ближайших игроков
- Выносливость и скорость

## Обучение агента

Используется DQN с адаптацией под SMDP — агент выбирает макро-действие и получает суммарную награду за всё время его выполнения:

```
R_t = r_{t} + γ·r_{t+1} + γ²·r_{t+2} + ... + γ^τ·r_{t+τ}
```

Вознаграждение складывается из:
- +100 за гол
- −100 за пропущенный гол
- +бонус за приближение к мячу
- +бонус за владение мячом
- −штраф за желтую/красную карточки

## Запуск

### Зависимости

- macOS (Apple M2) или Linux
- Python 3.14 + PyTorch
- RoboCup Soccer Simulator (`rcssserver >= 19.0`)
- helios-base с pybind11

### Сборка

```bash
cd helios-base
./bootstrap && ./configure
cd src/player && make -j4
```

### Обучение агента на позиции N

```bash
# 1. В swap_roles.py указать позицию агента
AGENT_B = "3"

# 2. Запустить смену формации
python3 scripts/swap_roles.py

# 3. Запустить обучение (50 матчей)
source .venv314/bin/activate
bash scripts/train_loop.sh 50
```

### Мониторинг

```bash
tail -f helios-base/src/logs/agent_2_steps.csv

# Построение графиков
python3 python_dqn/plot_logs.py
```
